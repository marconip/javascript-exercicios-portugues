DOM - Adiciona classes em diferentes modos

Adiciona classes diferentes para cada item clicado
Ao click, pega toda a lista e insere e retira class diferentes para cada item

Adiciona classe ao click e retira com outro click.
Ao click, pega toda a lista com "forEach" e insere e retira a mesma class (actives), para cada item com "classList.toggle"

Adicionar classe somente no menu clicado
Ao click, será adicionado uma classe (active) no menu clicado e clicando em outro removera a classe do antigo adicionando no outro botão.
Usar-se "forEach" para os botões, variavel para botões active + if length > 0 + botoes com className.replace + adiciona a class "active". replace.
