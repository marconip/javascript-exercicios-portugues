Handling Events - manipulação de eventos

Events - Executar um evento ao click
Display de alerta aparecerá ao clicar o botão (criado no html, botão uma função para chamar no JS)
Events - Executar um evento quando ao click
Foi escrito uma função "onclick" que pegando o elemento html já existente e clicando nele mostrará a data

onchange - Geralmente usado para fazer mudanças em caixa de textos
Foi escrito uma função "onchange" pega "onchange" já existente no html e clicando nele alterará os textos para maiusculos alterando o "value".
Event Listeners - manipulador de eventos, algumas formas de criar "click".

Exemplo 02 é o mesmo que o ex.01 a diferença é que é criado a função com "onclick = função" só que sem o remove.
