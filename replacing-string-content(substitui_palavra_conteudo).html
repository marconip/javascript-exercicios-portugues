String Methods - armazena e manipula

Replacing String Content - Mudar apenas uma palavra escolhida, no texto, sendo case sensitive.

Só muda o primeiro atributo palavra escolhida com mesmo nome.

A palavra que vi trocar será essa: " mudar" essa outra com mesmo nome mudar não troca.

Replacing Case Sensitive - Mudar palavra case sensitive com: /xxx/i.

Escrever o JS sem aspas.

A palavra que vi trocar será essa: " MUDAR".

Replacing as mesmas palavras - Mudar apenas as palavras com mesmo nome. mudar com: /xxx/g

A palavra que vai trocar será essa: " nova" essa outra com mesmo nome nova troca também.
